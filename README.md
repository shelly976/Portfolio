# 🌐 Portfolio Website

A personal portfolio website built using **React + Vite**.  
This project showcases different sections such as Home, Skills, Projects, and Contact using reusable React components and separate CSS files for styling.

---

# 🚀 Features

- Responsive Navigation Bar
- Home Section
- Skills Section
- Projects Section
- Contact Section
- Reusable React Components
- Separate CSS Styling
- Clean UI Design
- Fast Development using Vite

---

# 🛠️ Technologies Used

- React
- Vite
- JavaScript
- CSS

# ⚙️ Installation

## Clone Repository

```bash
git clone <repository-url>
```

---

## Install Dependencies

```bash
npm install
```

---

# ▶️ Run Development Server

```bash
npm run dev
```

The application will start on:

```bash
http://localhost:5173
```
